j+

A normal slideshow setup with no changes:

<div id="mySlider"> <img src="http://lorempixel.com/430/300/cats/1" src1="http://lorempixel.com/900/500/cats/1"/> <img src="http://lorempixel.com/430/300/cats/2" src1="http://lorempixel.com/900/500/cats/2"/> <img src="http://lorempixel.com/430/300/cats/3" src1="http://lorempixel.com/900/500/cats/3"/> </div> <script> $(document).ready(function(){ $('#mySlider').jplus(); }); </script>

Multiple static slideshows with varying colours, structure is the same as above (x4):

<script> $(document).ready(function(){ $('#mySlider1').jplus({ auto:false, color: '#60DCFF' }); $('#mySlider2').jplus({ auto:false, color: '#FF5E01' }); $('#mySlider3').jplus({ auto:false, color: '#000000' }); $('#mySlider4').jplus({ auto:false }); }); </script>

These are just 2 examples of how the plugin can be used and although only 3 images were used in the slideshows throughout the example - the amount of images you want to use is entirely up to you.
